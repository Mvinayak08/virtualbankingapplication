    Title

    Virtual Banking Application

    Discription

    Virtual Banking application is a client-server based web application. The project is designed using spring framework where maven is used as a building tool and project is initialized using spring boot. Java is used as primary programming language where Java 8 is used as run time environment.

    How does it work

    • The project is designed to perform following operations, Basically the operations are categorized into two types,
    1. Bank Manager side Operations
    2. Bank Customer side Operations


    • Bank Manager side operations contains:-
    1. Login
    2. View Bank Customer
    3. View List of Bank Customers
    4. Update Bank Customer as per requirement
    5. Delete Bank Customer
    6. Logout


    • Bank Customer side operations contains:-
    1. Registration
    2. Login
    3. View profile
    4. View balance
    5. Perform Transactions like credit or debit
    6. Check mini-statement
    7. Logout

    Project Specifications

    Specifications : Details

    Swagger

    About Swagger

    Swagger is a suite of tools for API developers from SmartBear Software and a former specification upon which the OpenAPI Specification is based. When creating APIs, Swagger tooling may be used to automatically generate an Open API document based on the code itself. This embeds the API description in the source code of a project and is informally called code-first or bottom-up API development. When described by an OpenAPI document, Swagger open-source tooling may be used to interact directly with the API through the Swagger UI. This project allows connections directly to live APIs through an interactive, HTML-based user interface. Requests can be made directly from the UI and the options explored by the user of the interface.
